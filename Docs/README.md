## About eShopOnWeb

eShopOnWeb is a sample ASP.NET Core reference application demonstrating a single-process (monolithic) application architecture and deployment model. It's designed to support the free .PDF download ebook: [Architecting Modern Web Applications with ASP.NET Core and Azure](https://aka.ms/webappebook).
